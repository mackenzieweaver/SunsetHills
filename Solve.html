The Solution

Sunset Hills

Enter in a number between 1 and 10 into each of the input fields. These numbers will represent how many floors are in each building. Since the sun sets in the west, the left-most building will always be able to see the sunset. The algorithm will tell you of the rest of the buildings which ones can see the sunset. If the left most building is the tallest, none of the others will be able to see.
